# 📦 API

一個使用 Node.js 建立的後端專案，包含基本頁面、表單送出（Contact）、API 串接與路由管理。

---

## 🚀 專案特色

* 🔹 Node.js + Express
* 🔹 SQLite 資料庫


---

## 📁 專案結構

```
src/
  api/                # API 請求
    contact.js
    server.js         # SQLite API
    todo.db           # SQLite 資料庫

```


## 🛠️ 安裝與啟動
---
* 套件 
```bash
npm install express sqlite3 cors
```
* 1️⃣後端啟動
```bash
 node server.js
```
* 2️⃣前端啟動 
```bash
npm run dev
```
